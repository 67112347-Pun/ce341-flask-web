สรุปความรู้เกี่ยวกับ Docker

Docker คืออะไร?

Docker เป็นแพลตฟอร์มสำหรับสร้างและรัน Container ซึ่งเป็นสภาพแวดล้อมแยกจากกัน ทำให้เราส่งแอปไปให้คนอื่นใช้งานได้ง่าย โดยไม่ต้องกลัวปัญหาเรื่องการติดตั้ง dependency ต่าง ๆ

Image และ Container

  • Image : แม่แบบ (template) ที่บอกว่าภายใน container มีอะไรบ้าง เช่น base image, โค้ด, dependencies
  • Container : instance ที่รันมาจาก image 1 อัน (รันจริง ๆ)

คำสั่งที่ใช้บ่อย

  • docker build -t username/ce341-flask-web:latest . – สร้าง image จาก Dockerfile ในโฟลเดอร์ปัจจุบัน
  • docker images – ดูรายการ image ที่มีในเครื่อง
  • docker run -d -p 5000:5000 username/ce341-flask-web:latest – รัน container แบบ background และ map port
  • docker ps – ดู container ที่กำลังรันอยู่
  • docker stop <CONTAINER_ID> – หยุด container
  • docker push username/ce341-flask-web:latest – ส่ง image ขึ้น Docker Hub

สรุปความรู้เกี่ยวกับ Git และ GitHub

Git คืออะไร?

Git เป็นระบบควบคุมเวอร์ชัน (Version Control System: VCS) ที่ใช้เก็บประวัติการแก้ไขไฟล์โค้ด สามารถย้อนกลับไปดู/กู้ไฟล์เวอร์ชันเก่า ๆ ได้ และทำงานร่วมกันหลายคนได้สะดวก

GitHub คืออะไร?

GitHub เป็นบริการที่ให้เราเอา Git repository ไปเก็บบน Cloud ทำให้แชร์โค้ดให้คนอื่นดู/โหลด หรือทำงานร่วมกันผ่าน Internet ได้ง่าย

คำสั่งพื้นฐานของ Git

  • git init : สร้าง Git repository ใหม่ในโฟลเดอร์ปัจจุบัน
  • git status : ดูสถานะไฟล์ที่เปลี่ยนแปลง
  • git add . : เพิ่มไฟล์ทั้งหมดเข้ามาใน staging area
  • git commit -m "ข้อความ" : บันทึกการเปลี่ยนแปลงเป็น 1 commit
  • git remote add origin <URL> : ผูก repo local กับ GitHub
  • git push -u origin main : ส่งโค้ดขึ้น GitHub
